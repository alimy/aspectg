### AspectG Tutorial

### Usage
```$bash
$ git clone https://github.com/alimy/aspectg
$ cd aspectg/module/examples
$ go generate
$ go build
$ ./examples
```

Or 

```$bash
$ git clone https://github.com/alimy/aspectg
$ cd aspectg/module/examples
$ go build -tags "AspectG"
$ go build
$ ./examples
```
